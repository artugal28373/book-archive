# Java — Reading Order

## Path: Beginner → Advanced

1. **Think Java 2** — Downey/Mayfield
   Gentle CS-flavored intro. Read first if you've never touched Java.

2. **Introduction to Programming Using Java (javanotes9)** — David Eck
   More comprehensive intro than Think Java. Covers OOP thoroughly.

## What's missing (PAID — must buy for advanced)

- **Effective Java (3e)** — Joshua Bloch ⭐⭐ (THE Java book — every senior Java dev has read it)
- **Java Concurrency in Practice** — Brian Goetz et al. ⭐ (canonical concurrency)
- **Modern Java in Action** — Urma/Fusco/Mycroft (Java 8+ idioms)
- **Core Java Vol 1 + 2** — Cay Horstmann (reference)
- **Spring in Action** — Walls (if doing Spring)

## External free resources
- **Java docs** — docs.oracle.com/en/java/
- **Baeldung tutorials** — baeldung.com (best free Java site)
- **MIT 6.005 Software Construction** — Java-focused, free OCW

## Implementation milestones
- [ ] Build a CLI tool with proper Maven/Gradle setup
- [ ] Write a small Spring Boot REST API
- [ ] Use Java streams + lambdas fluently (read Effective Java item 42-48)
- [ ] Build something with concurrent collections + ExecutorService
- [ ] Profile + tune JVM (heap size, GC settings) on a real workload

## Use case
Java is mostly used for: enterprise backends (Spring), Android (though Kotlin is winning), large-scale data systems (Spark, Hadoop, Kafka are JVM). Pick your direction after the language basics.
