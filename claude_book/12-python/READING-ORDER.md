# Python — Reading Order

## Path: Beginner → Advanced

1. **A Byte of Python** — Swaroop CH
   Shortest gentle intro. ~100 pages. Read in a weekend.

2. **Python for Everybody** — Charles Severance
   Slightly more comprehensive intro; pairs with the free Coursera course.

3. **Think Python 2** — Allen Downey
   Classic CS-flavored intro. Read if you want the foundations more carefully.

4. **Python Notes for Professionals** — GoalKicker
   Reference book. Browse for topics; don't read cover to cover.

5. **The Hitchhiker's Guide to Python** — Reitz et al.
   Best-practices, project structure, packaging, virtual envs. Important once you write more than scripts.

6. **Python 3 Patterns, Recipes and Idioms** — Bruce Eckel
   Advanced idioms, OOP patterns in Python.

## What's missing (worth buying)
- **Fluent Python (2e)** — Ramalho — the canonical advanced Python book
- **Effective Python** — Brett Slatkin

## Order rationale
- Beginners: Byte → Severance → Think Python in that order, or pick ONE.
- Once comfortable, read Hitchhiker's Guide for project structure.
- Eckel for deeper OOP/idioms.
