# Math — Reading Order

## Why this comes first
You cannot read ML/AI/DL papers without proof literacy + linear algebra + probability. Lock these in first.

## Path: Beginner → Advanced

### Step 1 — Proof literacy (start here)
1. **Book of Proof** — Hammack (`math-for-cs/`) — gentlest on-ramp to proof writing
2. **Discrete Mathematics: An Open Introduction** — Levin (`math-for-cs/`) — combinatorics, graphs, recursion

### Step 2 — Calculus (single + multivariable)
3. **OpenStax Calculus Vol 1** — limits, derivatives, integrals
4. **OpenStax Calculus Vol 2** — sequences, series, techniques of integration
5. **OpenStax Calculus Vol 3** — multivariable
6. **Active Calculus (Single Variable)** — Boelkins (alt or supplement to OpenStax)
7. **Active Calculus (Multivariable)** — Schlicker

### Step 3 — Linear Algebra
8. **Linear Algebra** — Hefferon (`linear-algebra/`) — friendly first pass
9. **Linear Algebra Lecture Notes (MIT 18.06)** — Strang — algorithmic intuition
10. **Linear Algebra Done Right** — Axler (4e) — proof-based, deeper

### Step 4 — Probability & Statistics
11. **OpenStax Introductory Statistics** — vocabulary, descriptive stats
12. **Think Stats 2** — Downey — applied stats with Python

### Step 5 — Discrete + Math for CS
13. **Mathematics for CS (MIT)** — Lehman/Leighton/Meyer — the canonical CS-flavored math text
14. **Foundations of Computer Science** — Aho/Ullman — broader CS theory

### Step 6 — Math for ML / Optimization (when ML-ready)
15. **Mathematics for Machine Learning** — Deisenroth/Faisal/Ong — exactly what ML needs
16. **Convex Optimization** — Boyd/Vandenberghe — read chapters 2, 3, 9 first

## Quickest path to ML readiness
If your goal is "can read ML papers ASAP":
Hammack → Hefferon → OpenStax Stats → Mathematics for ML → Boyd ch 2/3/9.
Skip Axler, FOCS, Active Calculus until later.
