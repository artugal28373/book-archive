# Books to Get for Each Category

You currently have **89 legally-free books**. This file lists what's still missing for a complete library, split into:

- **FREE** = legally free, you can download from the URL given
- **PAID** = you need to buy (or use O'Reilly/library subscription)

For paid books: cheapest legal options are usually (1) O'Reilly Learning subscription (~$49/mo, has all O'Reilly + many others), (2) your university library, (3) Kindle regional pricing (esp. India/SEA), (4) Humble Bundle tech book bundles.

# Priority shopping list (if you can buy ~$200 of books)

In order of ROI for a CS grad targeting **AI Engineer + SWE** roles:

1. **Designing Data-Intensive Applications** — Kleppmann (covers 03-DB, 06-sysdesign, 23-DataEng)
2. **Hands-On Machine Learning** — Géron (07-ML)
3. **Designing Machine Learning Systems** — Chip Huyen (21-MLOps)
4. **Build a Large Language Model from Scratch** — Raschka (22-LLM-eng)
5. **CLRS** or **Skiena** — pick one (16-Algos)
6. **The Pragmatic Programmer** — Hunt/Thomas (05-SWE)
7. **Clean Architecture** — R. Martin (20-Architecture)
8. **Effective Java** OR **A Tour of C++** — based on your language focus

Or get **O'Reilly Learning subscription ($49/month)** — includes 1, 2, 3, 5, 6, 7 for one month's price.
