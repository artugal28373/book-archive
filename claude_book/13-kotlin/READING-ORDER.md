# Kotlin — Reading Order

## Path: Beginner → Advanced

1. **Kotlin Notes for Professionals** — GoalKicker
   Quick concise tour of the language with examples.

2. **Kotlin Reference (official)** — JetBrains
   The canonical language reference. Use as both tutorial and lookup.

## What's missing (worth buying)
- **Kotlin in Action** — Jemerov/Isakova (paid) — the canonical Kotlin book
- **Atomic Kotlin** — Eckel/Isakova (paid) — best teaching style

## External free resources
- **Kotlin Koans** — kotlinlang.org/docs/koans.html — interactive exercises
- **JetBrains Academy Kotlin track** — partly free
- **Android with Kotlin** — developer.android.com/courses (free Google course)

## Reading order
1. GoalKicker for a 1-2 day overview
2. Official reference systematically (~1-2 weeks)
3. Kotlin Koans for hands-on practice
4. Buy Kotlin in Action when ready for depth

## Use case
Kotlin is mostly used for:
- **Android development** (replacing Java)
- **Server-side JVM** (Ktor, Spring with Kotlin)
- **Kotlin Multiplatform / Mobile (KMM)**

Pick your use case and find use-case-specific resources after the language basics.
