## Books worth buying (NOT free)

These are the gaps in any free CS library. Buy from publishers (Pearson, MIT Press, O'Reilly) — don't pirate. Or use an O'Reilly Learning subscription.

| Book | Author(s) | Why |
|---|---|---|
| Artificial Intelligence: A Modern Approach (4e) | Russell, Norvig | Canonical AI textbook |
| Designing Data-Intensive Applications | Kleppmann | Industry-standard distributed/DB book |
| CLRS — Introduction to Algorithms | Cormen, Leiserson, Rivest, Stein | Algorithms reference |
| Hands-On Machine Learning | Aurelien Geron | Best practical ML intro |
| Computer Architecture: A Quantitative Approach | Hennessy, Patterson | Standard advanced architecture |
| Computer Systems: A Programmer's Perspective | Bryant, O'Hallaron | Code → hardware bridge |
| The Pragmatic Programmer | Hunt, Thomas | SWE classic |
| Database System Concepts (Korth) | Silberschatz, Korth, Sudarshan | DB fundamentals |
| Quantum Computation & Quantum Information | Nielsen, Chuang | QC standard reference |
| Effective Modern C++ | Scott Meyers | Modern C++ in practice |
| Effective Java | Joshua Bloch | Same for Java |
| Kotlin in Action | Jemerov, Isakova | Best Kotlin book |

## Legal / piracy note

Everything in this folder was downloaded from the author's or publisher's official free release site. If a future you (or someone else) wants to update or re-download, use the URLs above. Do not download books from sites like LibGen, Z-Library, etc. — those are illegal in most jurisdictions and often serve malware.
