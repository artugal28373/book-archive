# Quantum Computing — Reading Order

## Prerequisites
- Linear algebra (vectors, matrices, eigenvalues, complex numbers, tensor product)
- Probability basics
- Comfortable with mathematical notation

## Path: Beginner → Advanced

1. **Quantum Computing: Lecture Notes** — Ronald de Wolf
   The cleanest CS-flavored intro. Start here.

2. **Lecture Notes on Quantum Algorithms** — Andrew Childs
   Algorithm-focused deep dive. Read after de Wolf.

3. **Preskill — Ph 219/229 Lecture Notes** (chapters 1–7, 10)
   The classic full treatment from Caltech. Read selectively after Childs.

4. **Quantum Information Theory** — Mark Wilde
   For the information-theoretic depth: entropy, channels, capacity. Heavy.

5. **Quantum Algorithm Implementations for Beginners** — Coles et al.
   Hands-on with Qiskit/Cirq examples.

### Quantum Machine Learning (after the basics)
6. **Quantum Machine Learning** — Biamonte et al. (Nature review)
   Survey of QML; read AFTER you have classical ML + basic QC down.

## What's missing (worth buying)
- **Quantum Computation and Quantum Information** — Nielsen & Chuang ("Mike & Ike", paid). The canonical reference.

## External resources
- **IBM Qiskit Textbook** (free HTML) — qiskit.org/textbook — for hands-on coding
- **Quantum Country** (free HTML, spaced-repetition) — quantum.country

## Realistic path
QC is a long road. Plan ~12-18 months from zero to "can read QC papers" with serious effort. You probably want classical ML solid FIRST before adding QML — most QML hype outpaces what's actually achievable on near-term hardware.
