# CS Library — Beginner to Advanced

A curated, **legally free** book library for a CS graduate focused on ML/AI/DL and SWE. Books are released free by their authors/publishers — no piracy.

## How to use this library

**Read in this order** for breadth:
1. **Math first** (01-math) — without this, ML/DL is fog. Hefferon LA → Mathematics for ML → Think Stats → Convex Optimization.
2. **Programming + DSA in parallel** (12, 14, 15, 16) — pick one language to get fluent (recommend Python for ML, C++ for systems), do all the algorithms.
3. **Systems track** (02, 03, 04) — OSTEP for OS, then networks, then databases.
4. **Build engineering judgment** (05, 06) — Pro Git is small but essential; system design is mostly external resources.
5. **ML/AI track** (07, 09, 11, 08) — ISLP → ESL → D2L → Bishop/Murphy → Sutton & Barto → SLP3.
6. **Specialize** (10 quantum, etc.) — pick depth based on research interest.
7. **Research methodology** (17) — read this *early*, not last.

## Important: books alone won't get you to research-advanced

These books cover ~80% of the foundations. The last 20% (modern DL, LLMs, distributed systems frontier) requires:
- arXiv papers (especially cs.LG, cs.CL, cs.DC)
- Conference proceedings: NeurIPS, ICML, ICLR, ACL, EMNLP, OSDI, SOSP, SIGMOD
- Recorded courses: Stanford CS224n (NLP), CS231n (CV), CS229 (ML), MIT 6.824 (distributed), CMU 15-445 (DBs)
- Implementing things from scratch (RL agents, transformers, B-trees, raft)

See `17-research-methodology/research-path.md` for a structured path.

## Books NOT included (paid, but worth buying)

| Field | Book | Why it matters |
|---|---|---|
| AI | Russell & Norvig — *AI: A Modern Approach* (4e) | The canonical AI textbook |
| Distributed | Kleppmann — *Designing Data-Intensive Applications* | Industry standard |
| Algorithms | Cormen et al. — *CLRS* | Reference for any algorithms research |
| ML/DL | Aurelien Geron — *Hands-On ML* | Best practical intro |
| Architecture | Hennessy & Patterson — *Computer Architecture* | Standard for advanced architecture |
| Systems | Bryant & O'Hallaron — *CS:APP* | Bridge between code and hardware |

Buy from the publisher or O'Reilly Learning subscription. Don't pirate.
